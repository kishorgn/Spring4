A simple demo about Spring MVC and Hibernate integration
 + One-to-many and Many-to-one relationship
 + Uses DAO Services and Transaction management
 + Used bidirectional relationship
 + Annotation configuration @ Entity level
 + Hibernate's SessionFactory has injected into DAO's.
 + Single <front_controler>-servlet.xml is the Spring configuration which contains 
    both Spring configuration and Hibernate SessionFactory configuration
